# Hi, I'm Saurabh 👋
Senior/Lead AI & Web3 Engineer · technical founder · 8+ years shipping production systems from 0→1 to scale.

I build **AI you can prove** — systems where *"where did this come from?"* has a span-level answer — and the **on-chain layer** that makes that proof verifiable: citation-grounded agents, audit-survivable provenance, and Solidity contracts that anchor and verify data on the blockchain.

My work spans **agentic RAG**, **on-prem SLM inference**, and **smart-contract / on-chain agents** — across legal, compliance, fintech, and enterprise document intelligence.
